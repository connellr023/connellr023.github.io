overview

This project was developed for the course CPSC 481, Human Computer Interaction. The goal of this project was to design and develop a food ordering interface for a restaurant. The restaurant we chose was Gogi, a Korean BBQ restaurant. The interface was designed to be used on a tablet device attached to the table.
