overview

Created for Calgary Hacks 2024, Atla serves as a dynamic hub for coordinators to effortlessly share events in preset categories. Users easily discover nearby events through an intuitive filtering system. Coordinators can have events posted within minutes, fostering community engagement.

challenges

Integrating a map library called react-leaflet with server-side rendering in Next.js posed unexpected challenges. Exploring Redis as an alternative database to something like PostgreSQL brought a unique learning curve, requiring mastery of unfamiliar CLI syntax.

acheivements

We take pride in seamlessly fetching data from Calgary's Open Data Portal and Redis database, dynamically loading it onto the map and webpage. Overcoming hurdles, including successfully implementing the interactive map, stands out as a significant achievement.
